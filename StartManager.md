# 启动脚本

``` shell
node socketServer/socketServer.js 服务端
```

``` shell
node socketClient/startClient.js 测试用客户端
```

``` shell
python camera/cameraWs.py 动捕摄像头转websocket
```

``` shell
node camera/simulated/wsServer.js 模拟摄像头转websocket
```

``` shell
node webGL/pointLog/pointLog.js 记录实验过程的点
```

``` shell
node webGL/pointLog/player.js websocket播放记录的点
```
