## 模拟客户端

模拟客户端在 /socketClient 文件夹

shell> node startClient.js

## 模拟服务端

模拟服务端在 /socketServer 文件夹

shell> node socketServer.js

## 模拟动捕相机

模拟动捕相机在 /camera/simulator 文件夹

shell> node wsServer.js
